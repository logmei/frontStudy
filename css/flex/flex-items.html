项目属性

flex：flex-grow,flex-shrink,flex-basis的简写，默认值为0 1 auto 后两个属性可选
两个快捷键 auto(1 1 auto) 和none(0 0 auto)
1、默认
2、第一个元素auto
3、第一个元素none
4、第一个元素1 0 auto
