媒体特性和响应式布局

1、设置media标签,告诉浏览器设备宽度为作为视图宽度并禁止初始的缩放。user-scalable=no能够解决ipad切换横屏之后需要触摸才能回到具体尺寸的问题

2、通过媒介查询来设置样式

@media screen and (max-width=980px){ }

3、设置多种视图宽度

/** iPad **/ @media only screen and (min-width: 768px) and (max-width: 1024px) {} /** iPhone **/ @media only screen and (min-width: 320px) and (max-width: 767px) {}

注意：

1、宽度使用百分比

2、处理图片缩放

(1)、(img的height设置为百分比不起作用)
(2)、指定最大宽度，实际宽度小于最大宽度是为默认大小，实际宽度大于最大宽度时缩小
(3)、用::before和::after伪元素和自定义属性
